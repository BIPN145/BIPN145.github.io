# Fly Tracking

The code in this folder will help you track your fly videos.

*If you have your videos stored in your Google Drive* you should use the [Colab version](https://bipn145.github.io/FlyTracking/ColabNotebook.html). Otherwise, if your videos are stored locally on the lab computer, use the [Local version](https://bipn145.github.io/FlyTracking/Script.html).
